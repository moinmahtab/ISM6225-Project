About Us

Hello! I'm Moin Mahtab, a student developing this website for my Master's program. This website was built using HTML, CSS, and JavaScript to demonstrate CRUD operations and data analytics. I do not have a lot of experience with website development. My website is a simple shoe sale website that provides classic Nike and Adidas shoe deals. User is able to add shoe orders in the Manage Inventory tab. The data analytics tab using JS, creates a bar chart that tracks the order quantity by brand. The inventory updates are visualized using Chart.js. I worked on some hover codes on images and texts to improve user interaction. In my free time, I enjoy sports, spending time with my family and analyzing data!
